# فلوسي فين — Flosy Fein

تطبيق Flutter محلي لإدارة الدخل والمصروفات والميزانيات والأهداف المالية، مع دعم العربية والإنجليزية.

## المزايا الحالية

- تسجيل الدخل والمصروفات وحفظها محليًا باستخدام Hive.
- ميزانيات شهرية حسب التصنيف.
- أهداف ادخارية وتحديات مالية.
- تحليل مبسط للمصروفات وسجل للعمليات.
- وضع فاتح وداكن ودعم العربية والإنجليزية.
- إعلانات تجريبية فقط باستخدام معرفات Google الرسمية للاختبار.

## التشغيل محليًا

```bash
flutter pub get
flutter run
```

## الفحص والاختبارات

```bash
flutter analyze
flutter test
flutter build apk --debug
```

تعمل هذه الأوامر أيضًا تلقائيًا داخل GitHub Actions عند فتح Pull Request على فرع `main`.

## إعداد AdMob

المشروع لا يحتوي على أي معرفات إعلانية إنتاجية. جميع App IDs وAd Unit IDs الحالية هي معرفات اختبار رسمية من Google.

قبل النشر الفعلي:

1. أنشئ التطبيق داخل حساب AdMob الصحيح.
2. استبدل App ID التجريبي في:
   - `android/app/src/main/AndroidManifest.xml`
   - `ios/Runner/Info.plist`
3. استبدل Ad Unit IDs التجريبية داخل `lib/core/utils/ad_helper.dart`.
4. أضف موافقة المستخدم وسياسة الخصوصية المطلوبة قبل تحميل إعلانات إنتاجية.

## Premium والمشتريات

ميزة Premium غير قابلة للشراء حاليًا. تم تعطيل زر الشراء التجريبي، ويجب ربط Google Play Billing وStoreKit والتحقق من إيصالات الشراء قبل السماح بتفعيل Premium.

## توقيع Android

يمكن بناء نسخة محلية أو CI بدون ملف توقيع إنتاجي؛ ستستخدم نسخة البناء مفتاح Debug مؤقتًا. لإصدار نسخة متجر، أنشئ `android/key.properties` واربطه بمفتاح Release الحقيقي. لا ترفع ملفات المفاتيح أو كلمات المرور إلى GitHub.
